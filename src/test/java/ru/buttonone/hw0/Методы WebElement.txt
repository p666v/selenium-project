
click() - Нажмите на этот элемент.

findElement(By by) - Найдите первое WebElement, используя данный метод:

  * findElement(By.id()) - Находит элементы, атрибут ID которых соответствует искомому значению.
  
  * findElement(By.name()) - Находит элементы, атрибут NAME которых соответствует искомому значению.
  
  * findElement(By.className()) - Находит элементы, имя класса которых содержит искомое значение (составные имена классов не допускаются).
  
  * findElement(By.tagName()) - Находит элементы, имя тега которых соответствует искомому значению.
  
  * findElement(By.linkText()) - Находит элементы привязки, видимый текст которых соответствует искомому значению.
  
  * findElement(By.partialLinkText()) - Находит элементы привязки, видимый текст которых содержит искомое значение. Если совпадают несколько элементов, будет выбран только первый.
  
  * findElement(By.xpath()) - Находит элементы, соответствующие выражению XPath.
  
  * findElement(By.cssSelector()) - Находит элементы, соответствующие селектору CSS.


findElements(By by) - Найдите все элементы в текущем контексте, используя данный механизм.

getAccessibleName() - Получает результат вычисления доступного имени и описания доступного имени элемента.

getAriaRole() - Получает результат вычисления роли элемента WAI-ARIA.

getAttribute(String name) - Получить значение заданного атрибута элемента.

getCssValue(String propertyName) - Получите значение данного свойства CSS.

getDomAttribute(String name) - Получить значение заданного атрибута элемента.

getDomProperty(String name) - Получить значение заданного свойства элемента.

getLocation() - Где на странице находится верхний левый угол отображаемого элемента?

getRect() - Расположение и размер отображаемого элемента 

getShadowRoot() - Представление теневого корня элемента для доступа к теневой модели DOM веб-компонента. 

getSize() - Какова ширина и высота отображаемого элемента?

getTagName() - Получите имя тега этого элемента.

getText() - Получите текст тега этого элемента.

isDisplayed()- Этот элемент отображается или нет? Этот метод позволяет избежать проблемы анализа атрибута «стиль» элемента.

isEnabled() - Включен ли элемент в данный момент или нет? Обычно это возвращает true для всего, кроме отключенных элементов ввода.

isSelected() - Определите, выбран ли этот элемент или нет.

sendKeys(CharSequence... keysToSend) - Используйте этот метод для имитации ввода элемента, который может установить его значение.

submit() - Если этот текущий элемент является формой или элементом внутри формы, он будет отправлен на удаленный сервер.
