
close() - Закройте текущее окно, выйдя из браузера, если это последнее открытое окно.

findElement(By by) - Найти WebElement, используя данный метод.

findElements(By by) - Найти все элементы на текущей странице, используя данный механизм.

get(String url) - Загрузите новую веб-страницу в текущем окне браузера.

getCurrentUrl() - Получите строку, представляющую текущий URL-адрес, который просматривает браузер.

getPageSource() -  Получите источник последней загруженной страницы.

getTitle() - Получить заголовок текущей страницы.

getWindowHandle() -  Верните непрозрачный дескриптор этого окна, который однозначно идентифицирует его в этом экземпляре драйвера.

getWindowHandles() - Возвращает набор дескрипторов окон, которые можно использовать для обхода всех открытых окон этого экземпляра WebDriver, передавая их в switchTo().WebDriver.Options.window()


manage() - Позволяет изменять настройки драйвера:

  * .addCookie(Cookie cookie) - Добавьте определенный файл cookie.

  * .deleteAllCookies() - Удалите все файлы cookie для текущего домена.

  * .deleteCookie(Cookie cookie) - Удалите файл cookie из «банки cookie» браузера.

  * .deleteCookieNamed(String name) - Удалите указанный файл cookie из текущего домена.

  * .getCookieNamed(String name) - Получите файл cookie с заданным именем.

  * .getCookies() - Получите все файлы cookie для текущего домена.

  * .logs() - Получает Logsинтерфейс, используемый для получения различных типов журналов.

  * .timeouts() - интерфейс управления таймаутами драйверов.

  * .window() - интерфейс управления текущим окном.



navigate() - Навигация:

  * .back() - Вернитесь на один «элемент» в истории браузера.

  * .forward() - Переместить один «элемент» вперед в истории браузера.

  * .refresh() - Обновить текущую страницу

  * .to(String url) - Загрузите новую веб-страницу в текущем окне браузера.

  * .to(URL url) - Перегруженная версия to(String)позволяет легко передавать URL-адрес.


quit() - Завершает работу этого драйвера, закрывая все связанные окна.

switchTo() - Отправляйте будущие команды в другой фрейм или окно.
