
* addArguments(String... arguments) - Добавляет дополнительные аргументы командной строки, которые будут использоваться при запуске Chrome.

    start-maximized: Максимизировать режим открытым
    Chrome incognito: Нет просмотра трассировки открытого браузера
    --headless: Главный режим (фон работает)
    --disable-extensions: Отключить существующие расширения на Chrome Browser
    --disable-popup-blocking: Отключить всплывающее окно
    make-default-browser: Установите Chrome в качестве браузера по умолчанию
    version: Печать Chrome Edition
    disable-infobars: Предотвратите Chrome от отображения "Chrome управляют автоматическим контролем программного обеспечения"
